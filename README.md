# PMG Programming Challenge Solution

Solution to PMG's csv-combiner. Command line argument passes several .csv files path. The script reads all the csv files and combines all the rows and also adds a 'filename' column indicating which file the row belongs to.

## csvcombiner.py

command line:
```
$ python ./csvcombiner.py ./fixtures_small/accessories.csv ./fixtures_small/clothing.csv > combined.csv
```
